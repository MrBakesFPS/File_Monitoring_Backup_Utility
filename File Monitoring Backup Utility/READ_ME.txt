Thank you for choosing File Monitoring Backup Utility!

Original idea by Jared Medeiros called "File Integrity Monitor"
Remastered by Tyson Koopman to include backing up monitored files

This program is intended to help you back up a project you're
working on, by monitoring the file directory you provide and
backing it up periodically based on the modification count
you provide as well.

If this is your first time using this program, I recommend
creating a test folder with at test .txt document inside
in order to see how the program works, then edit the .txt
while the program is running, saving after every change.

Go ahead and run this program once to see how it works,
then give it a shot when you work on your next project!
